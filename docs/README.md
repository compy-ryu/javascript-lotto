# 기능 목록

- 로또 금액을 입력하여 구매를 시도 수 있다.
  - 입력 예외사항
  - [x] 입력란이 빈칸일 수 없다.
    - '' => error
    - ' ' => error
  - [x] 숫자만 입력할 수 있다.
    - '천원' => error
  - [x] 천원 단위로 입력할 수 있다.
    - '1100' => error
- 입력된 금액만큼 로또를 구매한다.
  - [x] 로또번호는 1 ~ 45 사이의 숫자로 총 6개 번호가 저장된다.
    - [x] 로또번호는 중복될 수 없다.
    - [x] 로또번호는 숫자만 입력할 수 있다.
    - [x] 로또번호는 6개의 숫자를 가져야한다.
- [x] 사용자가 번호 보기 토클을 활성화하면 로또번호를 확인할 수 있어야 한다.
